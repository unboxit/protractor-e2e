__Setup__


`$ npm install -g protractor cucumber lr-http-server`

`$ webdriver-manager update`

`$ npm install -g yo generator-ng-component`

`$ npm install`


__Usage__

`$ npm run server`

`$ npm test` - for jasmine2

`$ npm run cucumber` - for cucumber
